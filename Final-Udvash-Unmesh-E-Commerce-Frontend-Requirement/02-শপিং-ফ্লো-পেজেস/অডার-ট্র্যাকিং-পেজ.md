# অর্ডার ট্র্যাকিং পেজ

## 📌 পেজের উদ্দেশ্য
সকল অর্ডারের তালিকা দেখা, নির্দিষ্ট অর্ডারের বর্তমান স্ট্যাটাস ট্র্যাক করা এবং রিটার্ন/রিফান্ড রিকোয়েস্ট করা।

> 🔗 **সংশ্লিষ্ট পেজ:**
> - আমার অর্ডার → [12-আমার-অর্ডার.md](../03-একাউন্ট-ম্যানেজমেন্ট-পেজেস/12-আমার-অর্ডার.md)

---

## ♻️ কেন্দ্রীয় লজিক যা সব পেজে প্রতিফলিত হবে

```
[অর্ডার কনফার্মেশন পেজ] → "অর্ডার ট্র্যাক করুন"
      ↓
লগড ইন? → হ্যাঁ → অর্ডার ট্র্যাকিং পেজ (নির্দিষ্ট অর্ডারের টাইমলাইন)
           না  → লগইন পেজ (?returnUrl=/my-orders)
                      ↓ লগইন শেষে
                 অর্ডার ট্র্যাকিং পেজ

[এসএমএস ট্র্যাকিং লিংক] → সরাসরি নির্দিষ্ট অর্ডারের পেজ

[হেডার → প্রোফাইল → "আমার অর্ডার"] → সব অর্ডারের তালিকা
```

---

## 🔐 অ্যাক্সেস লজিক
- এই পেজে প্রবেশ করতে **লগইন থাকতে হবে।**
- লগড আউট থাকলে → লগইন পেজ (`?returnUrl=/my-orders`)।

## 🔄 ইউজার ফ্লো

### ফ্লো ১ — কনফার্মেশন থেকে ট্র্যাকিং
1. পেমেন্ট সফল → কনফার্মেশন পেজে **"অর্ডার ট্র্যাক করুন"** ক্লিক।
2. অর্ডার ট্র্যাকিং পেজে নির্দিষ্ট অর্ডারের টাইমলাইন দেখাবে।

### ফ্লো ২ — প্রোফাইল থেকে অর্ডার হিস্ট্রি
1. হেডারের প্রোফাইল আইকন → "আমার অর্ডার"।
2. সব অর্ডারের তালিকা → নির্দিষ্ট অর্ডারে "বিস্তারিত দেখুন" ক্লিক।

### ফ্লো ৩ — এসএমএস ট্র্যাকিং লিংক
1. অর্ডার দেওয়ার পর এসএমএসে ট্র্যাকিং লিংক।
2. লিংকে ক্লিক → সরাসরি ওই অর্ডারের ট্র্যাকিং পেজ।

## 💡 বিশেষ লজিক

### রিটার্ন ও রিফান্ড ফ্লো
"রিটার্ন/রিফান্ড" ক্লিক করলে:
1. **কারণ নির্বাচন**: ভুল পণ্য / ক্ষতিগ্রস্ত / মান কম / আর দরকার নেই।
2. **বিবরণ** (অপশনাল): টেক্সটএরিয়া।
3. **ছবি আপলোড** (অপশনাল): সর্বোচ্চ ৫টি, প্রতিটি ৫ MB।
4. **রিফান্ড পদ্ধতি**: মূল মাধ্যমে ফেরত বা ওয়ালেট ক্রেডিট।
5. **সাবমিট** → টিকেট নম্বর পাবে।

### অর্ডার বাতিল লজিক
- শুধু **"প্রক্রিয়াধীন"** স্ট্যাটাসে বাতিল করা যাবে।
- শিপিং হয়ে গেলে → রিটার্ন প্রক্রিয়া ব্যবহার করতে হবে।
- বাতিল নিশ্চিত হলে → রিফান্ড ৩–৫ কার্যদিবসে।

### পুনরায় অর্ডার
- আগের পণ্যগুলো কার্টে যোগ হবে; স্টক নেই এমন পণ্য বাদ পড়বে এবং জানানো হবে।

### এম্পটি স্টেট
```
📦 এখনো কোনো অর্ডার করেননি
[কেনাকাটা শুরু করুন →]
```

| ডেস্কটপ | অর্ডার তালিকা বাম + ডিটেইলস ডান |

## 🎯 সাফল্যের মানদণ্ড

- অর্ডার স্ট্যাটাস রিয়েল-টাইমে আপডেট (WebSocket বা Polling)।
- ডেলিভারি টাইমলাইন স্পষ্ট ও বোধগম্য।
- প্রতিটি স্ট্যাটাসে সঠিক অ্যাকশন বাটন।
- রিটার্ন রিকোয়েস্ট সর্বোচ্চ ৩ স্টেপে সম্পন্ন।
- এসএমএস ট্র্যাকিং লিংক সরাসরি নির্দিষ্ট অর্ডারে নিয়ে যাবে।
