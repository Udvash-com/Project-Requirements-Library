### স্মার্ট সাজেশন সাইডবার (ডানে, স্টিকি)

গ্রাহক পেজে যা করছেন তার সাথে মিলিয়ে সাজেশন বদলাবে:

```
গ্রাহক যখন...              সাইডবারে দেখাবে...
──────────────────────────────────────────────────
ইমেজ দেখছেন               →  "এই ব্র্যান্ডের অন্য পোশাক"
রিভিউ পড়ছেন               →  "একই রেটিং এর পোশাক"
প্রাইস দেখছেন               →  "বান্ডল অফার / সাশ্রয়"
স্পেসিফিকেশন দেখছেন       →  "একই ধরনের পোশাক / ফ্যাব্রিক"
কার্টে যোগ করলেন           →  "এর সাথে কিনুন" (আপসেল)
```

**সাইডবারের স্ট্রাকচার:**

```
┌─────────────────────────────┐
│  💡 আপনার পছন্দ হতে পারে   │
│  ─────────────────────────  │
│  [মিনি কার্ড ১]               │
│  [মিনি কার্ড ২]               │
│  [মিনি কার্ড ৩]               │
│                             │
│  ─────────────────────────  │
│  🎁 বান্ডলে নিলে সাশ্রয়        │
│এই পোশাক + [ম্যাচিং ওড়না / প্যান্ট] = ৳২,৪৮০  │
│  ~~৳২,৫৫০~~   (৳৭০ সাশ্রয়)  │
│  [একসাথে কার্টে নিন]          │
└─────────────────────────────┘
```

> **কেন কার্যকর:** গ্রাহক পেজ স্ক্রল না করেই সাজেশন দেখতে পান — রিলেটেড পণ্য কেনার সম্ভাবনা ৩–৫ গুণ বাড়ে।

### "সিদ্ধান্ত হেল্পার" বক্স

প্রাইস সেকশনের ঠিক নিচে একটি ছোট সামারি বক্স:

```
┌──────────────────────────────────────────────────────┐
│  কেন এই পোশাকটি কিনবেন?                            │
│  ✅ ৫৪২ জন ব্যবহার করেছেন — ৪.৭ রেটিং               │
│  ✅ ২৬৩ জন যাচাইকৃত ক্রেতা রিভিউ দিয়েছেন             │
│  ✅ এই সপ্তাহে ১২৮ জন কিনেছেন                       │
│  ✅ ৭ দিনের রিটার্ন গ্যারান্টি                             │
└──────────────────────────────────────────────────────┘
```

> **কেন কার্যকর:** সামাজিক প্রমাণ (Social Proof) গ্রাহকের দ্বিধা কমায় এবং বিশ্বাসযোগ্যতা তৈরি করে।

### ইনলাইন সাজেশন (স্ক্রলের মাঝে)

শুধু পেজের শেষে নয়, কন্টেন্টের মাঝে মাঝে প্রাসঙ্গিক সাজেশন দেখানো হবে:

```
[বিবরণ সেকশন শেষ]
┌──────────────────────────────────────────────────────┐
│  👗 কে-ক্রাফট-এর এই পোশাকগুলোও পছন্দ হতে পারে      │
│  [পাঞ্জাবি]  [শার্ট]  [টি-শার্ট]  →                         │
└──────────────────────────────────────────────────────┘
[স্পেসিফিকেশন সেকশন]

[রিভিউ সেকশন শেষ]
┌──────────────────────────────────────────────────────┐
│  🎁 এই পণ্যটি কেনা মানুষরা এটিও কিনেছেন               │
│        [শাড়ি]  [সালোয়ার-কামিজ]  [ফতুয়া]  →           │
└──────────────────────────────────────────────────────┘
[ব্র্যান্ড ও স্টোর তথ্য]
```

> **কেন কার্যকর:** গ্রাহক যখন একটি সেকশন শেষ করেন, তখনই প্রাসঙ্গিক সাজেশন দেখলে ক্লিক করার সম্ভাবনা বেশি থাকে।

### স্পিড রিড সারাংশ (ভাঁজযোগ্য)

দীর্ঘ বিবরণের বদলে গ্রাহককে আগে সংক্ষিপ্ত দেখান:

```
বিবরণ সেকশন:
─────────────────────────────────────────────────────
  ✦ ক্যাটাগরি:           পাঞ্জাবি / সালোয়ার-কামিজ
  ✦ ব্র্যান্ড ও ম্যাটেরিয়াল: কে-ক্রাফট, ১০০% পিওর কটন
  ✦ সাইজ ও ফিট:         রেগুলার ফিট, ৪২-৪৮ সাইজ
  ✦ ক্রেতার মতামত:      "খুবই আরামদায়ক কাপড়, কালারও সুন্দর"
  ✦ ব্র্যান্ডের অন্যান্য পোশাক:  [শার্ট] [ফতুয়া] →  
  [▾ সম্পূর্ণ বিবরণ পড়ুন]
─────────────────────────────────────────────────────
```

> **কেন কার্যকর:** ব্যস্ত গ্রাহক দ্রুত সিদ্ধান্ত নিতে পারেন; আগ্রহীরা বিস্তারিত পড়তে পারেন।

### মোবাইলে ড্রয়ার-স্টাইল বিবরণ

মোবাইলে পুরো পেজ স্ক্রল না করে সেকশনগুলো ড্রয়ারে (Bottom Sheet) দেখানো হবে:

```
মোবাইল স্ক্রিন:
┌─────────────────────────┐
│  [ইমেজ গ্যালারি]         │
│                         │
│  মেনজ লিনেন পাঞ্জাবি     │
│  ৳১,৬৫০  ~~৳১,৯৫০~~   │
│  ⭐ ৪.৭  (৫৪২টি রিভিউ) │
│                         │
│  [বিবরণ ▾] [স্পেক ▾] [রিভিউ ▾]  ← ট্যাপে নিচ থেকে ড্রয়ার আসবে
│                         │
├─────────────────────────┤
│ ৳১,৬৫০  [কার্টে] [কিনুন] │  ← সর্বদা স্টিকি বটম বার
└─────────────────────────┘
```

> **কেন কার্যকর:** মোবাইল স্ক্রিনে অনেক স্ক্রল না করেও সব তথ্য পাওয়া যায়।

         ↕

         ↕
৪. [ইনলাইন সাজেশন: একই ব্র্যান্ডের পণ্য]       ← প্রাসঙ্গিক মুহূর্তে

         ↕

         ↕

## সাজেশন সেকশনের ৩টি স্তর (কোথায় কোনটি)

# প্রোডাক্ট স্পেসিফিক সেকশন 


- প্রোডাক্ট ইমেজ গ্যালারি (বাম এরিয়া)
- **মেইন ইমেজ ভিউয়ার**
  - **মেইন ইমেজ ভিউয়ার**: বড় উচ্চ-রেজোলিউশন প্রোডাক্ট ইমেজ। হোভার করলে ম্যাগনিফায়ার/জুম-ইন, মোবাইলে পিঞ্চ-জুম।
- **থাম্বনেইল স্ট্রিপ**
  - **থাম্বনেইল স্ট্রিপ**: ফ্রন্ট কভার, ব্যাক কভার, ইন্টেরিয়র পেজ স্যাম্পল (কয়েকটি পৃষ্ঠার প্রিভিউ) — নিচে বা বাম পাশে, বাম-ডান অ্যারো দিয়ে স্ক্রল।
- **ফুলস্ক্রিন মোড**
  - **ফুলস্ক্রিন মোড**: মেইন ইমেজে ক্লিক করলে ফুলস্ক্রিন মোডাল ওপেন হবে।
- **ইমেজ গণনা ইন্ডিকেটর**
  - **ইমেজ গণনা ইন্ডিকেটর**: "১ / ৫" — নিচে বা কোণে।
- **প্রোডাক্ট স্পেসিফিকেশন**
  - **প্রোডাক্ট টাইটেল** (H1, বোল্ড)
  - **ব্র্যান্ড নাম** — ক্লিকেবল (ব্র্যান্ড পেজে লিঙ্ক)
  - **টাইপ / ফিট**: রেগুলার ফিট / স্লিম ফিট
  - **অরিজিন / প্রস্তুতকারক**: দেশি / বিদেশি
  - **বারকোড / EAN**: 978-984-xxx-xxxx
  - **ডিজাইন কোড**: 984-xxx-xxxx
  - **পিস / সেট**: থ্রি-পিস / ২ পিস / সিঙ্গল
  - **প্যাকেজিং টাইপ**: বক্স / পলি প্যাক
  - **ডাইমেনশন / সাইজ / বহর**: ৪৬ ইঞ্চি / ৪৮ ইঞ্চি
  - **ফ্যাব্রিক / ম্যাটেরিয়াল**: সুতি / সিল্ক / জর্জেট
  - **ফ্যাব্রিক কোয়ালিটি / সুতার কাউন্ট**: ১০০% পিওর কটন
  - **বেস কালার**: ব্লু / ব্ল্যাক / রেড
  - **ডিজাইন / এমব্রয়ডারি কালার**: গোল্ডেন / মাল্টিকালার
  - **কালেকশন / সিজন**: ঈদ কালেকশন / সামার প্রোমো
  - **টার্গেট অডিয়েন্স**: Men / Women / Kids
  - **উৎস / মেইড ইন**: বাংলাদেশ (Local) / ইমপোর্টেড
  - **ওজন**: ৩৫০ গ্রাম
  - **কালার ও সাইজ ভ্যারিয়েন্ট**: ব্লু (M, L) / ব্ল্যাক (XL)
  - **ক্যাটাগরি ট্যাগ** (ক্লিকেবল): শাড়ি • পাঞ্জাবি • কটন
  - **SKU / প্রোডাক্ট কোড** (ছোট ধূসর টেক্সট)
  - **ভ্যারিয়েন্ট বাটনস**: `[M]` `[L]` `[XL]` `[XXL]`
  - **স্পেশাল এডিশন**: একাধিক এডিশন থাকলে `[ডিজাইনার এডিশন]`
- **প্রোমো কোর্ড**
  - **প্রোমো কোর্ড**: মাল্টিপল প্রোমো কোড কালেক্ট বাটন।
- **প্রাইস সেকশন**
  - **মূল প্রাইস** (ধুসর, ~~1000/-~~)
  - **বর্তমান প্রাইস** (বড় এবং হাই কন্ট্রস্ট, 900/-)
  - **ডিসকাউন্ট ব্যাজ** (লাল/কমলা ব্যাজ): "10% ছাড়" বা "৳100 সাশ্রয়"
