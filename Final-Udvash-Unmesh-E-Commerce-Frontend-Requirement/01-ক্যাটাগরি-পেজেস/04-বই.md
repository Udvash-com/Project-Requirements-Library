# বুক ডিটেইলস পেজ 

## 📌 পেজ এর উদ্দেশ্য
> **দৃষ্টিভঙ্গি:** গ্রাহক কার্ড ক্লিক করে এসেছেন — তিনি ইতিমধ্যে আগ্রহী। এখন তার কাজ হলো **সিদ্ধান্ত নেওয়া।** পেজের প্রতিটি সেকশন সেই সিদ্ধান্তকে সহজ করবে।

### স্মার্ট সাজেশন সাইডবার (ডানে, স্টিকি)

গ্রাহক পেজে যা করছেন তার সাথে মিলিয়ে সাজেশন বদলাবে:

```
গ্রাহক যখন...              সাইডবারে দেখাবে...
──────────────────────────────────────────────────
ইমেজ দেখছেন               →  "এই লেখকের অন্য বই"
রিভিউ পড়ছেন               →  "একই রেটিং এর বই"
প্রাইস দেখছেন               →  "বান্ডল অফার / সাশ্রয়"
স্পেসিফিকেশন দেখছেন       →  "একই জনরা / জেনারের বই"
কার্টে যোগ করলেন           →  "এর সাথে কিনুন" (আপসেল)
```

**সাইডবারের স্ট্রাকচার:**

```
┌─────────────────────────────┐
│  💡 আপনার পছন্দ হতে পারে   │
│  ─────────────────────────  │
│  [মিনি কার্ড ১]               │
│  [মিনি কার্ড ২]               │
│  [মিনি কার্ড ৩]               │
│                             │
│  ─────────────────────────  │
│  🎁 বান্ডলে নিলে সাশ্রয়        │
│  এই বই + [অন্য বই] = ৳৪৮০  │
│  ~~৳৫৫০~~   (৳৭০ সাশ্রয়)    │
│  [একসাথে কার্টে নিন]          │
└─────────────────────────────┘
```

> **কেন কার্যকর:** গ্রাহক পেজ স্ক্রল না করেই সাজেশন দেখতে পান — রিলেটেড পণ্য কেনার সম্ভাবনা ৩–৫ গুণ বাড়ে।

# প্রোডাক্ট স্পেসিফিক সেকশন 


- প্রোডাক্ট ইমেজ গ্যালারি (বাম এরিয়া)
- **মেইন ইমেজ ভিউয়ার**
  - **মেইন ইমেজ ভিউয়ার**: বড় উচ্চ-রেজোলিউশন বুক কভার ইমেজ। হোভার করলে ম্যাগনিফায়ার/জুম-ইন, মোবাইলে পিঞ্চ-জুম।
- **থাম্বনেইল স্ট্রিপ**
  - **থাম্বনেইল স্ট্রিপ**: ফ্রন্ট কভার, ব্যাক কভার, ইন্টেরিয়র পেজ স্যাম্পল (কয়েকটি পৃষ্ঠার প্রিভিউ) — নিচে বা বাম পাশে, বাম-ডান অ্যারো দিয়ে স্ক্রল।
- **ফুলস্ক্রিন মোড**
  - **ফুলস্ক্রিন মোড**: মেইন ইমেজে ক্লিক করলে ফুলস্ক্রিন মোডাল ওপেন হবে।
- **ইমেজ গণনা ইন্ডিকেটর**
  - **ইমেজ গণনা ইন্ডিকেটর**: "১ / ৫" — নিচে বা কোণে।
- **প্রোডাক্ট স্পেসিফিকেশন**
  - **প্রোডাক্ট টাইটেল** (H1, বোল্ড)
  - **প্রোডাক্ট ডেসক্রিপশন:** ১০ হাজার ক্যারেক্টার পর্যন্ত হতে পারে।
  - **পৃষ্ঠা সংখ্যা**: ২৮০ 
  - **ক্যাটাগরি**: HSC প্যারালাল টেক্সট
  - **ভার্সন**: বাংলা, ইংরেজি
  - **সংস্করণ / এডিশন**: একাধিক এডিশন থাকলে `[৫ম সংস্করণ]` `[বিশেষ সংস্করণ]`
  - 🔄 রিটার্ন: ক্রয়ের ৭ দিনের মধ্যে রিটার্ন/এক্সচেঞ্জ
- **প্রাইস সেকশন**
  - **মূল প্রাইস** (ধুসর, ~~1000/-~~)
  - **বর্তমান প্রাইস** (বড় এবং হাই কন্ট্রস্ট, 900/-)
  - **ডিসকাউন্ট ব্যাজ**: "10% ছাড়" বা "৳100 সাশ্রয়"
  - **অফার টাইমার** (যদি লিমিটেড টাইম অফার থাকে): "⏱ অফার শেষ হচ্ছে: ০২:৩৪:১২"

- **নেভিগেশন বাটনস**
  - **কোয়ান্টিটি**:  [-]  ১  [+]
  - **"একটু পড়ুন" বাটন**: বই ক্যাটাগরিতে বিশেষ বাটন — ক্লিক করলে বইয়ের প্রথম কয়েকটি পৃষ্ঠার প্রিভিউ মোডাল ওপেন হবে।
   - **এখনই কিনুন**: সরাসরি চেকআউট/পেমেন্ট পেজে নিয়ে যায়।
  - **কার্টে যোগ করুন**: কার্টে অ্যাড হয়, পেজে থাকে, মিনি-কার্ট নোটিফিকেশন দেখায়।
  - **স্টক আউট/প্রি-অর্ডার/কামিং সুন হলে**: প্রোডাক্ট ইমেজের উপর একটি ওয়াটার মার্ক দেখাবে। স্টক আউট হলে "স্টক আউট" লেখা দেখাবে, প্রি-অর্ডার হলে "প্রি-অর্ডার" লেখা দেখাবে, কামিং সুন হলে "কামিং সুন" লেখা দেখাবে। 
  - **প্রি-বুকিং মোড** (সেলার চালু রাখলে): "⏳ প্রি-বুকিং করুন" বাটন।
  - **অর্ডার টাইপ**:
    - **অর্ডার টাইপ**: রেগুলার / প্রি-অর্ডার / কামিং সুন
  - **এখনই কিনুন (Buy Now)**: প্রাইমারি অ্যাকশন
  - **কার্টে যোগ করুন**: সেকেন্ডারি অ্যাকশন
